Linguagens

  • HTML5
    HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar e estruturar páginas da web. Ela é a base fundamental da maioria dos sites e fornece a estrutura básica para o conteúdo exibido em um navegador da web.
  • CSS3
    CSS3 (Cascading Style Sheets) é uma linguagem de estilo utilizada em conjunto com o HTML para definir a aparência e o layout das páginas da web. Ela oferece uma variedade de recursos e propriedades para estilizar elementos HTML e controlar sua apresentação visual.
  • JavaScript
    JavaScript é uma linguagem de programação de alto nível, orientada a objetos e baseada em scripts. Ela é amplamente utilizada para desenvolvimento web e permite a criação de páginas interativas, aplicações web complexas e muitos outros recursos dinâmicos.

  • C# / CSharp
    C# (pronuncia-se "C sharp") é uma linguagem de programação moderna e orientada a objetos, desenvolvida pela Microsoft. Ela faz parte do framework .NET e é amplamente utilizada para o desenvolvimento de aplicativos de desktop, jogos, aplicações web e dispositivos móveis.

  • SqlServer
    SQL Server é um sistema de gerenciamento de banco de dados relacional desenvolvido pela Microsoft. Ele oferece um ambiente seguro e escalável para armazenar, acessar e gerenciar grandes volumes de dados.
